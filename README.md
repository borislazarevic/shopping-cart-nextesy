# shopping-cart-nextesy
## Tech stack
- React
- Redux Toolkit
- React Router
- React Toastify: notification lib.
- Vite: builder tool.
- TypeScript
- Sass

## Quick Start 🏃‍♂️

### Install all dependencies
`npm install`

### Start the development server
`npm run dev`
- Open your browser and go to `http://localhost:5173/`
